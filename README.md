# La Biblioteca de Alejandría

Llevo demasiado tiempo estudiando y ahora ya pierdo el orden de mi propia bibliografía. Así que estoy haciendo esto para tener mi propia video-documentación de todos los temas que sé y que me interesan.

Si me conoces o simplemente te interesa, para instalar y correr el programa ejecuta:

```bash
git clone https://github.com/sergioCaceresMC/LaBibliotecaDeAlejandria
cd LaBibliotecaDeAlejandria
npm run dev
```
