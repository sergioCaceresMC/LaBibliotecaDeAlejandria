## ¿Qué es flask?

Flask es un [framework](https://www.youtube.com/watch?v=xmHwYc-6W3E) que sirve para desarrollar páginas web. Es de los más básicos, tiene funciones bastante sencillas de entender y es muy didactico porque tienes completo control del código.

### Ventajas

- Es sencillo
- La documentación inicial es accesible
- Hay tutoriales de yt buenos

### Desventajas

- No tiene incorporado demasiadas herramientas, por ejemplo: ORM, login, jwt
- Puede generar varios conflictos de dependencias
- Se necesita una estructura clara y no es recomendable para proyectos grandes

## ¿Cuando sugiero flask?

A momento de escribir esto, yo no soy un experto del desarrollo web, aún estoy aprendiendo cosas de ciberseguridad, despliegue y controles de acceso. Para ello prefiero usar flask, porque a diferencia de otros frameworks como django, aquí tengo que configurar todo, así que me veo obligado a investigar y aprender.

**Resumen:** Ideal si estás aprendiendo
